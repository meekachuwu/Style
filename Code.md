# Conventions
## JavaScript
### Names
- Default:		1 word consisting of only lowercase letters
- Directories:	Capitalized
### Structure
- Code:			Modular & Object-Oriented
- Scope:		No global variables/functions, hardcoded values stored externally in serialized data formats
### Placement
- Indention:	Tabs
- Comments:		Same line as the code being described
- Operators:	Horizontally align when adjacent, same type, & same nesting depth
### Syntax
- Declarations:	`let`/`const`
- Conditions:	`switch`
- Functions:	`=>`
- Loops:		`foreach`
- Quotes:		``` ` ```
### Settings
- Directive:	`use strict`

## Reference
### Example
```
`use strict`;

const config = import(`Data/config.yaml`)

# W.I.P

```
